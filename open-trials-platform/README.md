# OpenTrials Platform

A full-stack prototype for adaptive analytics workflows on messy business datasets.

## What this repo contains

- Backend API for dataset ingestion and analytics
- Frontend dashboard for dynamic data exploration
- Early-stage architecture built for rapid iteration and SaaS migration

## How to use

1. Start the frontend with `npm run dev` from the repo root.
2. Review the product story in the top-level `README.md`.
3. Explore the backend in `open-trials-platform/backend/app`.

> The root README is the primary project narrative. This nested README is intentionally lightweight to avoid duplicate guidance.
